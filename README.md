
# First Python

1. mkdir first-python = 建資料夾

2. cd first-python = 進去

3. echo '...' > main.py = 寫 code 進檔案

4. python main.py = 執行

5. git init = Git 初始化

6. git add main.py = 加入追蹤

7. git commit -m "..." = 提交

8. git branch -M main = 改分支名

9. git remote add origin ... = 綁定 GitHub

10. git config user.email ... = 設定 email

11. git commit --amend ... = 修正作者

12. git push -u origin main = 推上 GitHub
